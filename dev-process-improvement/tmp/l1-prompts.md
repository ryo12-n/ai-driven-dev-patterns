# L1 マネージャーセッション プロンプトテンプレート

## 新規施策を開始する

```
initiatives/_template/ をコピーして initiatives/[施策名]/ を作成してください。
以下の要件で 00_proposal.md を記入してください。

【背景・課題】
（ここに記入）

【目標】
（ここに記入）

【スコープ】
（ここに記入）

その後、01_plan.md と 02_tasks.md も続けて作成してください。
```

## L2レポートを受けてゲート判定する

```
initiatives/[施策名]/ の以下のファイルを読んでください:
- 04_work_report.md（作業レポート）
- 06_eval_report.md（評価レポート）
- 07_issues.md（課題一覧）

01_plan.md の成功基準と照合し、08_gate_review.md にフェーズゲート判定を記入してください。
判定は「通過 / 条件付き通過 / 差し戻し」の3択で、具体的な根拠を添えてください。
```

## 計画を修正する

```
initiatives/[施策名]/ の 08_gate_review.md の判定結果に基づき、
01_plan.md と 02_tasks.md を修正してください。

修正理由と変更箇所を明記してください。
```

## 施策一覧の状況確認

```
initiatives/ 配下の全施策フォルダを確認し、
以下の形式で一覧を出力してください:

| 施策名 | 現在フェーズ | 最終更新 | 未解決課題数 | 次アクション |
```
