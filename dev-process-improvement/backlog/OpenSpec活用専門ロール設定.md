# OpenSpec 活用専門ロール設定

## メタ情報

| 項目 | 内容 |
|------|------|

| **ステータス** | 候補 |
| **起票日** | 2026-03-03 |

## 課題・背景

OpenSpec の skills を扱う専門ロールが存在しない。現状では OpenSpec の運用知識が各ロールに分散しており、ISS-006（ロール定義と OpenSpec 開発ライフサイクルの統合が未定義）の具体的な解決策が不足している。専門ロールを新設し、マネージャーとの Q&A フローを整備することで OpenSpec の活用品質を向上させる。

## 期待効果

- OpenSpec skills の運用品質向上（専門知識の集約）
- ISS-006 の具体的な解決策の一つ
- マネージャー経由の壁打ちフロー整備により、ユーザーとの Q&A が体系化

## 補足・参考情報

- ISS-006（ロール定義と OpenSpec 統合未定義）との関連
- inbox 原文: 「オープンスペックのskillのみを扱う専門のロールを作成したい。例えばskillの中にユーザーとの壁打ちが必要な内容があれば、マネージャーセッションへユーザーに聞くよう依頼するか、もしくは、オープンスペック専門ロールの作業履歴の中に、Q&Aのような資料用意して、マネージャーと、そこでやりとりする(マネージャーはユーザーと壁打ちしながらQ&Aへ回答を記載していく)などのフローを整備したい」
