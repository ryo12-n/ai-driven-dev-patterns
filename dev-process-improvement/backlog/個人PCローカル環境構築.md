## 課題・背景

現在 Claude Code Web 環境で開発を行っているが、1セッション=1リポジトリの制約（ISS-027）により、マルチリポ操作（リポジトリ分離施策の新リポ作成等）が実行不可能。また gh CLI も未インストール（ISS-026）で GitHub API 操作がブロックされている。ローカル PC（WSL）に Claude Code CLI 環境を構築することで、これらの制約を回避しつつ、iPhone（CodeSail）からのリモート指示にも対応できるようにする。

## 期待効果

- ISS-026/027 の解消。リポジトリ分離施策の再開が可能に
- マルチリポ操作対応（`--add-dir` オプション）
- iPhone からの開発指示が可能に（CodeSail 連携）
- gh CLI 利用可能。PR 作成等の自動化が復活

## 補足・参考情報

- 環境構成: WSL（Ubuntu）+ Claude Code CLI + CodeSail + VS Code（Remote-WSL）
- inbox に詳細な手順書あり（フェーズ2 on WSL 構成）
- 直接的にリポ分離施策（`dev-process-improvementリポジトリ分離`）のブロック解消に寄与
