# コラボレーションログ

トリアージで蒸留され、ルール・docs に反映されたエントリは削除する（git履歴が証跡）。

## フォーマット

```
### YYYY-MM-DD <セッションの主テーマ>
- **気づき**: （何が分かったか・何が変わったか）
- **蒸留候補**: （どのルール/ドキュメントに反映すべきか）
- **ステータス**: 未蒸留 / 蒸留済
```
