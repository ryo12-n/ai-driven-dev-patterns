# TG-008 チェック基準のルール変更連動更新

## 背景

`refine-triage-template` 施策で判明した知見。TG-008 はルール定義のポリシーチェックを行うメタルールだが、対象ルール（例: 課題起票フロー）が変更された場合、TG-008 自体のチェック基準も更新しないと旧基準で判定してしまう。

実際に triage-worker.md の TG-008 セクション内に旧フロー記述（「CSV 直接起票」）が残存し、評価者によって発見された。

## 提案

テンプレート変更施策のタスクリストに「TG-008 チェック基準の連動更新」を標準タスクとして含めるルールを検討する。

## 情報源

- `initiatives/refine-triage-template/08_gate_review.md` 必須把握事項 #1
- `initiatives/refine-triage-template/06_eval_report.md` ルール化候補 #1
