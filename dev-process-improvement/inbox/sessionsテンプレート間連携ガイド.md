# sessions/_template/ 間の連携関係ガイド

## 起源

施策「マネージャーロール整備」の 08_gate_review.md 次施策候補より。

## 提案内容

`sessions/_template/` 配下の4ファイル（plan.md, log.md, issues.md, report.md）間のデータフロー関係を明示するガイドを作成する。

## 背景

plan.md の完了条件 → log.md の進捗管理 → issues.md の課題管理 → report.md の知見セクション、という連携関係が暗黙的。テンプレート間の参照関係を明示すれば、新規セッション作成時の理解が容易になる。L2-worker と L2-evaluator の両方から同様の観察が報告された。
