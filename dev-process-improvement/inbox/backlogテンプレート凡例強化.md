# backlog テンプレート凡例強化・フィールド必須化

## 背景

「backlogを俯瞰できる一覧資料作成」施策で以下の問題が発見された。

1. **優先度表記の不統一**: 🟢低（テンプレート凡例にない emoji）、emoji なし「中」が混在。テンプレート凡例は 🔴高/🟡中/🔵低 だが強制力がない
2. **対象リポジトリの欠損**: `claude-directory最適化(ai-driven-dev-patterns).md` と `triageテンプレートのブラッシュアップ.md` で未記載

## 提案

- `backlog/_template.md` の凡例に「必ず Unicode emoji（🔴🟡🔵）を使用すること」と明記する
- `対象リポジトリ` を必須フィールドとして明記する
- トリアージ承認時にフィールド完全性をチェックするバリデーションルールを追加する

- 施策: `initiatives/backlogを俯瞰できる一覧資料作成/`
- 04_work_report.md ルール化候補 #1, #2
- 06_eval_report.md ルール化候補 #1
