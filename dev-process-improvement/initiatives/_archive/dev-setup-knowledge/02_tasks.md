### T-001 詳細指示

調査対象:
- `.claude/settings.json` の `allowedTools` に記載のコマンド
- `dev-process-improvement/refs/ai-driven-development-poc/scripts/` 以下の全スクリプト
- `dev-process-improvement/refs/ai-driven-development-poc/.claude/` 以下の全ファイル
- リポジトリルートの `README.md`
- `.gitignore`（除外対象から逆引きで使用環境を推測）

調査方法: `grep` / `which` / ファイル読み取りを活用すること

### T-002 詳細指示

`docs/dev-setup.md` に含めるべき項目:
1. 前提OS（Linux/macOS）
2. 必須ツール一覧（ツール名・用途・最低バージョン・インストールコマンド）
3. 推奨ツール一覧（なくても動くが推奨）
4. セットアップ後の確認コマンド（`scripts/setup.sh` を実行する旨）
5. トラブルシューティング（よくある環境起因エラーと対処）
6. 更新履歴

### T-003 詳細指示

`scripts/setup.sh` の仕様:
- 実行権限: `chmod +x` 済みで commit すること
- 動作: 各ツールの有無と最低バージョンを確認し、不足している場合は `ERROR: <ツール名> が見つかりません` を出力
- 全ツールが揃っていれば `✅ セットアップ確認 OK` を出力して終了コード 0
- 不足があれば終了コード 1 で終了
- インストールは自動実行しない（案内のみ）
