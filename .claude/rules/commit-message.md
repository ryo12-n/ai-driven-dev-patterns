# コミットメッセージルール

## ベースルール

`roles/_base/common.md` セクション5「コミット規約」に従う（Conventional Commits 形式）。

## トリアージセッション固有のルール

トリアージセッションのコミットには以下の形式を使用する。

```
[triage] YYYYMMDD: <要約>
```

- `YYYYMMDD` はトリアージセッションのフォルダ名と一致させる
- 要約は作業内容を簡潔に記述する

### 例

```
[triage] 20260304: inbox 走査・バックログ整理
[triage] 20260304: 課題管理CSV レビュー・ルール整合性チェック
[triage] 20260304: トリアージレポート作成
```

## セッションロール別プレフィックス一覧

| プレフィックス | 用途 |
|--------------|------|
| `[triage]` | トリアージセッション |
| `[L1]` | 施策マネージャー（dev_manager） |
| `[L2-worker]` | 実装ワーカー |
| `[L2-evaluator]` | 評価ワーカー |
